# Game Engine

A miniature game engine built for EECS 404/498 (Section 07).

## Overview

Hi, I am Noah. This project started as a text-based C++ game and evolved into a lightweight cross-platform game engine.

As the course progressed, the engine gained support for:
- Images, audio, mouse, and touchpad input via SDL2, SDL2_image, SDL2_mixer, and SDL2_ttf
- Data-driven gameplay by replacing hardcoded logic with Lua-based scripting and templates
- Physics and event-based gameplay through Box2D and an integrated event manager

## Platform Support

The engine was developed with macOS, Windows, and Linux support in mind, and later expanded to Android.

Android-focused work includes:
- Touch panel controls
- Automatic scaling for phones and tablets
- APK export in addition to desktop builds
